# Starter Build

A mod that allows you to configure an initial build, primarily intended for use with MegabonkRandom (https://github.com/radsi/Megabonk-Random)

# How to install

Download [Bepinex](https://builds.bepinex.dev/projects/bepinex_be/752/BepInEx-Unity.IL2CPP-win-x64-6.0.0-be.752%2Bdd0655f.zip) and extract everything from the zip in the game's root directory
Then download the release of this repo and place the .dll inside BepInEx/plugins directory
